# material-texture-namer

Blender 上のマテリアルとテクスチャの名前を整理する単独 Claude スキル。Blender MCP 経由で動作します。

各マテリアルの**ベースカラー・テクスチャ名をマテリアル名に一致させる**のが目的です。複数マテリアルで共有されているテクスチャは**複製**し、必ずマテリアルごとに同名のテクスチャがあるようにします（不変条件：処理後はどのベーステクスチャも2つのマテリアルで共有されない）。

## 2つのモード

スキルは最初に必ずモードを尋ねます。

1. **名前一致のみ** — マテリアル名はそのままに、ベーステクスチャ名をマテリアル名へ一致（共有は複製）。
2. **名前一致＋わかりやすくリネーム** — 加えて、長く分かりにくいマテリアル名を短い半角英数字へ付け直し、テクスチャも同名に。TOON は `<名前>_toon`、共有 matcap / sphere 等はそのまま残置。

## 使い方

1. Blender を起動し Blender MCP アドオンを接続。
2. モデルを読み込む（MMD/VRoid なら Principled BSDF 変換前後どちらでも可）。
3. このスキルを呼び出すと、モード選択 → inspect → ドライラン（確認）→ 適用 → 検証 の順で進みます。

破壊的なジオメトリ変更はありませんが、データブロック名を変更し画像を複製することがあるため、実行前の .blend 保存を推奨します。
